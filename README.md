# Flappy Bird - Scratch

My first ever game! Built using Scratch for CS50 Week 0.

## How to play
- Press SPACE to make bird jump
- Avoid the pipes
- Try to get highest score!

## What I learned
- Variables and clones in Scratch
- Gravity simulation using velocity
- Collision detection
- Score system
